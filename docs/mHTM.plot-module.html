compute_axis_bounds(x, multiplier=1)

source code 

Compute the bounds for the axis. It is assumed that the data starts at 0, unless the data goes below zero. The bounds are based off the standard deviation.

Parameters:
  • x - The data to apply to the axis. If multiple values are supplied, the largest bounds are used.
  • multiplier - The multiplier to scale by. Recommended is 6 for Y axis and 1/6 for X axis.

compute_err(x, x_med=None, axis=1)

source code 

Compute the relative errors based off IQR.

Parameters:
  • x - A 2D numpy array containing the data across multiple iterations. Each row represents a unique instance of the data.
  • x_med - A 1D numpy array representing the median values.
  • axis - The axis around which to compute the data.
Returns:
A 2D numpy array containing the -y and +y errors.

plot_epoch(y_series, series_names=None, y_errs=None, y_label=None, title=None, semilog=False, out_path=None, show=True, start_epoch=1)

source code 

Basic plotter function for plotting various types of data against training epochs. Each item in the series should correspond to a single data point for that epoch.

Parameters:
  • y_series - A tuple containing all of the desired series to plot.
  • series_names - A tuple containing the names of the series.
  • y_errs - The error in the y values. There should be one per series per datapoint. It is assumed this is the standard deviation, but any error will work.
  • y_label - The label to use for the y-axis.
  • title - The name of the plot.
  • semilog - If True the y-axis will be plotted using a log(10) scale.
  • out_path - The full path to where the image should be saved. The file extension of this path will be used as the format type. If this value is None then the plot will not be saved, but displayed only.
  • show - If True the plot will be show upon creation.
  • start_epoch - The epoch to start plotting at.

plot_error(x_series, y_series, series_names=None, y_errs=None, x_label=None, y_label=None, title=None, semilog=False, out_path=None, show=True, xlim=None, ylim=None, ax=None, legend=True)

source code 

Basic plotter function for plotting data with error bars.

Parameters:
  • x_series - A tuple containing all of the desired series to plot.
  • y_series - A tuple containing all of the desired series to plot.
  • series_names - A tuple containing the names of the series.
  • y_errs - The error in the y values. There should be one per series per datapoint. It is assumed this is the standard deviation, but any error will work.
  • y_label - The label to use for the x-axis.
  • y_label - The label to use for the y-axis.
  • title - The name of the plot.
  • semilog - If True the y-axis will be plotted using a log(10) scale.
  • out_path - The full path to where the image should be saved. The file extension of this path will be used as the format type. If this value is None then the plot will not be saved, but displayed only.
  • show - If True the plot will be show upon creation.
  • xlim - A tuple containing the bounds to use for the x-axis; otherwise, if None, it is automatically scaled. If False the default is used.
  • ylim - A tuple containing the bounds to use for the y-axis; otherwise, if None, it is automatically scaled. If False the default is used.
  • ax - The axis to use.
  • legend - Boolean denoting if a legend should be made.

plot_image(x, shape, title=None, out_path=None, show=True)

source code 

Plot a grayscale image. It is assumed that the image has been normalized (0, 1).

Parameters:
  • x - A numpy array containing the image.
  • shape - The shape of the data. This should be a tuple of x and y dimensions.
  • title - The name of the plot.
  • out_path - The full path, including the extension, of where to save the plot. If this paramter is None, the image will not be saved.
  • show - If True the image will be shown.

plot_compare_images(x_series, shape, title=None, series_labels=None, out_path=None, show=True)

source code 

Construct a grid of grayscale images. It is assumed that the images have been normalized (0, 1). The images will be plotted row-wise based off the order they were provided.

Parameters:
  • x_series - A sequence of NumPY arrays containing a set of images.
  • shape - The shape of the data. This should be a tuple of x and y dimensions.
  • title - The name of the plot.
  • series_labels - The labels for each image in each series. If None, no labels will be made.
  • out_path - The full path, including the extension, of where to save the plot. If this parameter is None, the image will not be saved.
  • show - If True the image will be shown.
